## 🔧 API CHANGES

### **New Method: `isRechargeable()`**

**Signature**:
```cpp
bool isRechargeable() const;
```

**Returns**: 
- `true` for all 14 rechargeable techs
- `false` for ALKALINE (primary only)

**Usage Example**:
```cpp
BatteryModel nimh(TECH_NIMH, 8, 2500.0f);
BatteryModel alkaline(TECH_ALKALINE, 4, 2800.0f);

if (nimh.isRechargeable()) {
    // Can use charging API
    float v_abs = nimh.getAbsorptionVoltage();
} else {
    // Alkaline: discharge-only monitoring
    float soc = alkaline.ocvToSoc(measured_voltage, temp);
}
```

### **Updated Method: `getNominalVoltage()`**

Added cases:
```cpp
case TECH_NIMH:     return 1.2f * n_cells;
case TECH_ALKALINE: return 1.5f * n_cells;
```
